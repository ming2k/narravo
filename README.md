# Narravo

A powerful, minimalist browser extension for text-to-speech using Azure's Speech API. Built with React and the WXT framework.

## Key Features

- **High-Quality Voices**: Leveraging Azure's Neural TTS for natural-sounding speech.
- **Real-Time Streaming**: Audio starts playing instantly using the MediaSource API.
- **Smart Context Menu**: Right-click selected text on any page to read it aloud.
- **Modern UI**: Clean popup and setup pages with dark mode support.
- **Multi-Browser**: Native support for Chrome and Firefox.
